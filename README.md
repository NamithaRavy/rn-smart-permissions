# rn-smart-permissions

Permissions Library for React Native ( Android and iOS)

## Installation

```sh
npm install rn-smart-permissions
```

## Usage

```js
import { multiply } from 'rn-smart-permissions';

// ...

const result = await multiply(3, 7);
```
